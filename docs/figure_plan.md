# 図追加プラン

## 方針

- 定義だけで抽象度が高いスライドには, 対応する集合・函数・測度の概念図を置く.
- 既存の GIF は手順や近似の動きを見せる用途として残し, 新規 PNG は関係図・条件図・静的イメージを補う.
- 章扉や結論スライドには, 内容の再整理に効く場合だけ図を置く.
- 暗色モードでも読めるよう, 図は明るい紙面背景のカードとして `slide-figure` の枠内に表示する.
